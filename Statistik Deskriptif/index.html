    Statistika Deskriptif¶

    Statistika Deskriptif¶

    Pengertian :¶

    Statistika Deskriptif ialah metode representasi keseluruhan himpunan data spesifik dengan memberikan ringkasan pendek dan ukuran data. Statistika Deskriptif hanya terbatas dalam menyajikan dalam bentuk tabel, diagram, grafik, dan besaran lain. Sedangkan statistik inferensial selain mencakup statistik deskriptif juga dapat digunakan dalam melakukan estimasi dan penarikan kesimpulan terhadap populasi dari sampelnya, untuk sampai pada penarikan kesimpulan statistik inferensial melalui tahap uji Hipotetis dan uji Statistik.

    Tipe Data Statistika Deskriptif¶

    Mean( Rata-rata )¶

    Mean ialah sebuah nilai yang jumlah dari seluruh angka atau data dan dibagi banyak data.

    Misal memiliki N data bisa dihitung dengan Rumus sbb: $$ \bar x ={\sum \limits_{i=1}^{n} x_i \over N} = {x_1 + x_2 + x_3 + ... + x_n \over N} $$ Keterangan:

    • x = Rata - rata
    • a = Nilai ke N
    • n = Banyak Nilai atau Data
    Modus¶

    Modus ialah Nilai yang sering muncul. Jika kita tertarik pada data frekuensi, jumlah dari nilai dari kumpulan data, maka kita menggunakan modus. Modus baik digunakan pada skala ketegorik yaitu nominal atau ordinal. $$ M_o = Tb + p{b_1 \over b_1 + b_2} $$ Keterangan:

    • Mo = Modus dari kelompok data

    • Tb = Tepi bawah dari elemen modus

    • b1 = Selisih frekuensi antara elemen modus dengan elemen sebelumnya

    • b2 = Selisih frekuensi antara elemen modus dengan elemen sesudahnya

    • p = Panjang interval

    Nilai b1 dan b2 -> adalah mutlak ( always positive )

    Median¶

    Median ialah cara untuk menentukan letak tengah data setelah data disusun menurut urutan nilainya. Simbol untuk median yaitu Me.

    Median ialah nilai data tengah, dalam data kelompok memiliki rumus yang sama dengan mencari Q2 (Kuartil 2) seperti misalnya bisa anda lihat dibawah ini: $$ Me=Q_2 =\left( \begin{matrix} n+1 \over 2 \end{matrix} \right), jika\quad n\quad ganjil $$

    $$ Me=Q_2 =\left( \begin{matrix} {xn \over 2 } {xn+1\over 2} \over 2 \end{matrix} \right), jika\quad n\quad genap $$

    Keterangan

    • Me = Median dari data kelompok
    • n = banyak data
    Varians¶

    Varians ialah suatu ukuran penyebaran data, yang diukur dalam pangkat dua dari selisih data terhadap rata-ratanya. $$ \sigma^2 = {\sum \limits_{i=1}^{n} (x_i - \bar x)^2 \over n} $$ Keterangan :

    • xi = Titik data
    • x bar = Rata-rata dari semua titik data
    • n = Banyak dari anggota data
    Quartile¶

    Quartile ialah irisan nilai dari hasil pembagian data menjadi empat bagian yang sama besar. Nilai-nilai dari quartile biasanya dilambangkan dengan Q1 untuk quartile bawah. Q1 ini mempunyai nilai 25% dari data. Q2 mempunyai nilai sama seperti median yaitu 50% dari data. dan Q3 sebagai quartile atas mempunyai nilai 75% dari data. Dalam mencari quartile kita dapat menggunakan rumus berikut ini: $$ Q_1 = (n + 1) {1\over 4} $$

    $$ Q_2 = (n + 1) {1\over 2} $$

    $$ Q_3 = (n + 1) {3\over 4} $$

    Keterangan:

    • Q = Nilai dari Quartile
    • n = Banyak dari himpunan data

    Standar Deviasi

    Standar deviasi ialah merupakan ukuran dispersi kumpulan data relatif terhadap rata-rata. Standar deviasi dihitung dengan mengakar kuatdratkan nilai varians. Jika titik data lebih dari rata-rata dalam kumpulan data maka, semakin tinggi standar defiasi.

    Rumus: $$ \sigma^ = \sqrt {{\sum \limits_{i=1}^{n} (x_i - \bar x)^2 \over n}} $$

    Skewness¶

    Skewness / Kemiringan adalah ketidaksimetrisan pada suatu distribusi statistik dimana kurva tampak condong kekiri atau kekanan. Skewness digunakan untuk menentukan sejauh mana perbedaan suatu distribusi normal grafik muncul seperti kurva beebentuk lonceng. Ketika suatu distribusi mengalami kemiringan kesebelah kanan dan ekor disisi kanan kurva lebih panjang dari ekor sisi kiri kurva maka situasi ini dikatakan kemiringan positif dan sebaliknya dikatakan kemiringan negative.

    Rumus: $$ Skewness = {\sum \limits{i=1}^n (x_i - \bar x)^i \over (n- 1) \sigma^3} $$ Keterangan:

    • Xi = Titik data
    • X bar = Rata-rata dari distribusi
    • n = Jumlah titik dalam distribusi
    • o = Standar deviasi

    Berikut Penerapan Statistik Deskriptif di Python¶

    Sebelum itu buat data random melalui exel lalu kita import file tadi seperti dibawah ini:

    import pandas as pd
from scipy import stats
df = pd.read_csv('hamid.csv',sep=';')
data={"stats":['Min','Max','Mean','Standart Deviasi','Variasi','Skewnes','Quantile 1','Quantile 2',
               'Quantile 3','Median','Modus']}
for i in df.columns:
    data[i] = [df[i].min(), df[i].max(), df[i].mean(), round(df[i].std(),2), round(df[i].var(),2),
               round(df[i].skew(),2), df[i].quantile(0.25),df[i].quantile(0.5),df[i].quantile(0.75), 
               df[i].median(), stats.mode(df[i]).mode[0]]
tes = pd.DataFrame(data)
tes.style.hide_index()
